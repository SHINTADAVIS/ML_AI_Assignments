# ML & AI Practice Assignments

This repository contains my practice assignments and hands-on projects completed during my Machine Learning and Artificial Intelligence course.   
The notebooks cover various core concepts including recommendation systems, neural networks, NLP, classification, and clustering.  

## Contents  
1. Movie Recommendation System  
   A recommendation system built to suggest movies based on user preferences using machine learning techniques.  

2. Magic Gamma Telescope Project (Neural Network and Hyper parameter Tuning)  
   Implementation of neural networks with hyperparameter tuning for classifying gamma and hadron events.  

3. NLP Classification of News Articles  
   Natural Language Processing project to classify news articles into different categories.  

3. Spam Classification Different Approaches  
   Comparison of multiple machine learning approaches for spam email detection.  

4. Whole sale customers Clustering  
   Unsupervised learning project using clustering techniques to segment wholesale customers.
